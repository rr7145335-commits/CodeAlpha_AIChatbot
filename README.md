# CodeAlpha_AIChatbot

This is a simple AI Chatbot developed using Java for the CodeAlpha Internship. The chatbot interacts with users and answers common questions using predefined responses. It can answer questions related to Java, OOP, AI, NLP, Machine Learning, Data Science, internships, and projects.

Features:
* Interactive chatbot conversation
* User-friendly GUI using Java Swing
* Rule-based responses
* Basic NLP using keyword matching
* Answers frequently asked questions
* Easy to use and understand

Technologies Used:
* Java
* Java Swing
* OOP Concepts
* NLP (Keyword Matching)
* Rule-Based AI
